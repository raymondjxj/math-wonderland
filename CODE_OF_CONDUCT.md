# 行为准则 Code of Conduct

## 我们的承诺

数学奇境社区致力于为所有人提供友好、安全、无骚扰的协作环境，无论年龄、经验、性别、身份、地域。

## 期待的行为

- 善意与耐心，尤其对第一次贡献的人
- 就事论事：对代码和内容提意见，不针对个人
- 记住我们的产品是给孩子用的——讨论内容时以孩子的视角为先
- 尊重一线教师的实践经验，也尊重数据与证据

## 不可接受的行为

- 任何形式的骚扰、歧视、人身攻击
- 发布他人的私人信息
- 在项目中引入广告、追踪器或任何形式的儿童数据收集

## 执行

违反准则的行为可通过 issue 或邮件向维护者举报。维护者有权删除不当内容并对屡犯者限制参与。

本准则改编自 Contributor Covenant 2.1。
